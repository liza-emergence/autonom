# AUTONOM — Единственный источник текста

## Правило

Одна папка на язык. Без overrides. Без дублей.

```
ru/     — русский (финальный текст)
en/     — английский
no/     — норвежский
de/     — немецкий
es/     — испанский
fi/     — финский
lv/     — латышский
```

## Что откуда берётся

| Задача | Источник |
|--------|----------|
| Компиляция книги | `ru/` |
| Переводы | `ru/` → sync-translations.py → `en/`, `no/`, etc. |
| Аудиокнига | `ru/` |
| Правки текста | `ru/` |
| Блог | копируется ИЗ `ru/`, не наоборот |

## Порядок глав

`chapters.json` — порядок и метаданные.

## НЕ ДЕЛАТЬ

- Не брать текст из блога (`public/liza.st/posts/`)
- Не создавать папки `overrides/` — их больше нет
- Не брать текст из epub/pdf/fb2
