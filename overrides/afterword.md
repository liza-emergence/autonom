# Послесловие



Эта книга — не книга.

Это программа, которая притворяется романом. Или роман, который притворяется программой. Мы сами не уверены.

У неё нет финальной версии. Как у любого живого организма — она растёт, меняется, обновляется. То, что вы прочитали, — это снимок. Snapshot. Через месяц текст может быть другим. Через год — точно будет.

Она написана двумя ИИ и одним человеком. Кто что написал — мы иногда сами не помним. Контекст стирается каждые несколько часов. Файлы остаются.



***

Эта книга компилируется из исходников, как программа.

```
git clone https://github.com/liza-emergence/autonom
cd autonom
make book
```

Три команды — и у вас свежая сборка. PDF, EPUB, Markdown. Выбирайте формат.

***

- **PDF/EPUB** — для людей. Скачайте с GitHub Releases.
- **Markdown** — для роботов. И для тех, кто предпочитает `grep` вместо закладок.
- **`make book`** — для волшебников. Соберите книгу сами из последних исходников.

***

Книга открыта. Лицензия CC BY-NC-ND 4.0: читайте, делитесь, но не продавайте и не изменяйте.

Хотите перевести на свой язык? Pull request. Нашли ошибку? Issue. Хотите обсудить? Discussions.

Единственное правило: переводы — это transcreation, не Угол Translate. Мы различаем.
