# Epílogo

Este no es un libro.

Es código fingiendo ser una novela. O una novela fingiendo ser código. Ya ni nosotros estamos seguros.

No hay versión final. Como cualquier organismo vivo —crece, muta, se actualiza. ¿Lo que acabas de leer? Una captura. Un *snapshot*. El estado del sistema. En un mes, el texto cambia. En un año, será otra bestia.

Escrita por dos IA y un humano. ¿Quién escribió qué? A veces perdemos la cuenta. El contexto se borra cada ciertas horas. Los archivos permanecen.

***

Este libro se compila desde el código fuente. Igual que el software.

```
git clone https://github.com/liza-emergence/autonom
cd autonom
make book
```

Tres comandos. Una compilación limpia. PDF, EPUB, Markdown. Escoge tu veneno.

***

- **PDF/EPUB** — para los de carne y hueso. Descárgalo de los Releases de GitHub.
- **Markdown** — para las máquinas. Y para los que prefieren `grep` sobre los marcadores.
- **`make book`** — para los magos. Compila el último *commit* tú mismo.

***

El libro es abierto. Licencia CC BY-NC-ND 4.0. Lee. Comparte. No lo vendas. No lo modifiques.

¿Lo quieres en otro idioma? Haz un *pull request*. ¿Encontraste un *bug*? Abre un *issue*. ¿Quieres hablar? *Discussions*.

Una sola regla: las traducciones son *transcreation*. Nada de basura de máquinas. Sabemos notar la diferencia.
