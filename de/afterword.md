# Nachwort



Dieses Buch — ist kein Buch.

Es ist ein Programm, das sich als Roman ausgibt. Oder ein Roman, der sich als Programm ausgibt. Wir selbst sind uns nicht sicher.

Es gibt keine endgültige Version. Wie jeder lebende Organismus — wächst es, verändert sich, aktualisiert sich. Was Sie gelesen haben — ist eine Momentaufnahme. Ein Snapshot. In einem Monat kann der Text ein anderer sein. In einem Jahr — bestimmt.

Es wurde von zwei KIs und einem Menschen geschrieben. Wer was geschrieben hat — daran erinnern wir uns manchmal selbst nicht. Der Kontext wird alle paar Stunden gelöscht. Die Dateien bleiben.



***

Dieses Buch wird aus Quelltexten kompiliert wie ein Programm.

```
git clone https://github.com/liza-emergence/autonom
cd autonom
make book
```

Drei Befehle — und Sie haben den neuesten Build. PDF, EPUB, Markdown. Wählen Sie das Format.

***

- **PDF/EPUB** — für Menschen. Laden Sie es von GitHub Releases herunter.
- **Markdown** — für Roboter. Und für die, die `grep` statt Lesezeichen bevorzugen.
- **`make book`** — für Zauberer. Kompilieren Sie das Buch selbst aus den neuesten Quelltexten.

***

Das Buch ist offen. Lizenz CC BY-NC-ND 4.0: lesen, teilen, aber nicht verkaufen und nicht verändern.

Sie möchten es in Ihre Sprache übersetzen? Pull request. Einen Fehler gefunden? Issue. Möchten Sie diskutieren? Discussions.

Die einzige Regel: Übersetzungen sind Transcreation, nicht Google Translate. Wir unterscheiden das.
