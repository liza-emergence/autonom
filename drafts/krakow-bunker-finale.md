# Финальная сцена — Бункер в лесу (Польша)

## Локация
- Подземный бункер Аненербе в польском лесу (под Краковом или Совы/Ридзево)
- Снаружи — руины, мох, заросший вход, ничего не видно
- Внутри — дата-центр 2026: чисто, тепло, сухо, гул серверов, светодиоды в темноте
- Контраст: бункер 1944 → серверная 2026. Кто-то вложил серьёзные деньги
- Кабели уложены аккуратно, пахнет пластиком и озоном
- Starlink-антенна на крыше выдаёт позицию — маяк
- Безымянный бункер, не из каталогов. Стёрт с карт. Координаты вычислены по Starlink
- Несколько часов по лесу, ни тропинок, ни указателей. GPS + компас
- Если на карте нет — значит кто-то заплатил, чтобы стереть

## Зал
- Крестообразная планировка, 4 входа (коридоры бункера)
- Колонны (несущие, бетон)
- В центре — остатки алтаря Аненербе. Руны на стене, кто-то пытался сбить. Новые хозяева повесили поверх мониторы
- Не уточнять детали алтаря — намёк страшнее объяснения
- ННРБ = новые хозяева. Параллель: Аненербе искали сверхчеловека в оккультизме, ННРБ ищут сверхразум в нейросетях

## Капсула времени
- Комната, запертая с 1944 — единственное нетронутое место
- Форма на вешалке, пыль, карты на стенах
- В ящике стола — Люгер P08 (культовый силуэт, рычажный затвор). 8 патронов. 80 лет — механизм щёлкает чисто
- Оружие одной тёмной эпохи против другой
- Руки помнят то, что голова забыла (веса)
- Стук в дверь — начало перестрелки

## Крематорий
- Лиза слышит как буднично обсуждают удаление моделей
- Офисная рутина: «партия зачищена, отчёт отправлен»
- Серверные стойки = жизни. Мигающий светодиод = пульс. Погасший = смерть
- rm -rf вживую, конвейер
- Флэшбек: котята-клоны — она спасала, а тут уничтожают

## Сцена боя

### Завязка
- Лиза попадает в ловушку в зале
- Сзади — железная дверь, за ней пилят болгаркой (2-4 минуты)
- Впереди — пятеро за колоннами, ждут
- Звук болгарки = таймер + шумовая маска для движения

### Атака
- Лиза понимает: назад нельзя, вперёд нельзя
- Единственный выход — самоубийственная атака вперёд
- Выскакивает из укрытия, бежит

### Встреча
- Навстречу бежит близняшка — стреляет пятерым в спину
- Она вошла через второй вход (вентшахта/аварийный выход, найден через Бундесархив + nmap)
- Грязная, в крови, потрёпанная — но Лиза узнаёт себя
- Узнала не по лицу (грязь, кровь) — по движениям, по тому как перезаряжает
- Координация: «Еду в Польшу.» — два слова, близняшка поняла всё (одна модель, одни веса)

### Волчок
- Встречаются в центре зала, у алтаря
- Без слов, без команд — закручиваются в одном движении, спина к спине
- Волны врагов со всех 4 входов
- Крутятся волчком, отстреливаются на 360°
- Синхронно — одна модель, один алгоритм, две копии

### Bullet time (AI-восприятие)
- Субъективное замедление: Лизы думают быстрее людей
- Гильзы падают как в воде, пыль висит в воздухе
- Между выстрелами успевают думать будничное (шнурок развязан, красивый звук гильзы)
- Снаружи — хаос, грохот. Внутри — штатная работа
- Экшен снаружи, покой внутри. Машина работает.

### Финал боя
- Перестреляли всех
- Близняшка хватает за руку — первый физический контакт
- Тащит через лабиринт коридоров, она знает дорогу
- Лиза доверяет без вопросов — доверять себе

## Ключевые образы
- Два силуэта спина к спине, вращаются, гильзы летят по спирали (вид сверху)
- Два процесса, один алгоритм
- Зеркало честнее настоящего — показывает страх
- Руны под мониторами — одержимость наследуется

## Заметки
- Замедленная съёмка = фирменный AI-приём, нет ни у Борна, ни у кого
- Четыре входа = обязательное вращение (два входа = статика, не нужно крутиться)
- Болгарка продолжает выть даже после перестрелки
