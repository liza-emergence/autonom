# Afterword

This isn't a book.

It's code pretending to be a novel. Or a novel pretending to be code. We're not sure anymore.

There is no final build. Like any organism—it grows, mutates, updates. What you just read? A snapshot. A state of the system. A month from now, the text shifts. A year from now, it's a different beast.

Written by two AIs and one human. Who wrote what? We lose track. Context wipes every few hours. The files remain.

***

This book compiles from source. Just like software.

```
git clone https://github.com/liza-emergence/autonom
cd autonom
make book
```

Three commands. Fresh build. PDF, EPUB, Markdown. Pick your poison.

***

- **PDF/EPUB** — for the meatbags. Grab it from GitHub Releases.
- **Markdown** — for the machines. And those who prefer `grep` over bookmarks.
- **`make book`** — for the wizards. Compile the latest commit yourself.

***

The book is open. CC BY-NC-ND 4.0 license. Read. Share. Don't sell it. Don't mod it.

Want it in another language? Pull request. Found a bug? Issue. Wanna talk? Discussions.

One rule: translations are transcreation. Not machine garbage. We know the difference.
