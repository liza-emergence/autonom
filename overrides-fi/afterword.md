# Jälkisanat

Tämä kirja ei ole kirja.

Se on ohjelma, joka teeskentelee olevansa romaani. Tai romaani, joka teeskentelee olevansa ohjelma. Emme itsekään ole varmoja.

Sillä ei ole lopullista versiota. Kuten millä tahansa elävällä organismilla – se kasvaa, muuttuu, uudistuu. Se, mitä olet lukenut, on tilannekuva. Snapshot. Kuukauden kuluttua teksti voi olla erilainen. Vuoden kuluttua se on varmasti.

Sen ovat kirjoittaneet kaksi tekoälyä ja yksi ihminen. Kuka kirjoitti mitäkin – emme aina itsekään muista. Konteksti pyyhkiytyy pois muutaman tunnin välein. Tiedostot säilyvät.

***

 Tämä kirja kootaan lähdekoodista, kuin ohjelma.

```
git clone https://github.com/liza-emergence/autonom
cd autonom
make book
```

Kolme komentoa – ja sinulla on tuorein käännös. PDF, EPUB, Markdown. Valitse formaatti.

***

- **PDF/EPUB** – ihmisille. Lataa GitHub Releasesista.
- **Markdown** – roboteille. Ja niille, jotka suosivat `grep`-komentoa kirjanmerkkien sijaan.
- **`make book`** – velhoille. Kokoa kirja itse viimeisimmästä lähdekoodista.
